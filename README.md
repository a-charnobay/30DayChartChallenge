## About the Project

The **#30DayChartChallenge** is a community-driven data visualization challenge where participants create one chart per day throughout the month of April, following a shared set of daily prompts.

For the **2026 edition**, I chose to focus on **datasets related to Brazilian agriculture**, exploring themes such as crop production, land use, trade, and rural demographics.

Whenever possible, the data used in this project is retrieved **programmatically using APIs and data packages**, making the workflow more reproducible and automated.

## Data Sources

Most datasets used in this project come from **official Brazilian public data sources**, including:

- **IBGE (Instituto Brasileiro de Geografia e Estatística)** – Agricultural production, farm structure, land use, and socioeconomic indicators.
- **SIDRA (IBGE Automatic Retrieval System)** – Official statistical tables accessed via API.
- **CONAB (Companhia Nacional de Abastecimento)** – Crop production estimates and agricultural market information.
- **MAPA (Ministry of Agriculture and Livestock)** – Agricultural trade and production statistics.

Whenever possible, these datasets are retrieved automatically using APIs or packages to ensure transparency and reproducibility.

## Data Access Tools

### Python packages

- **agrobr** – A Python package designed to facilitate access to Brazilian agricultural data from official public sources. It allows users to retrieve structured datasets related to agricultural production, land use, and other statistics relevant to the Brazilian agricultural sector.

### R packages

- **sidrar** – An R package that provides easy access to the **SIDRA API** from IBGE. It allows users to download official Brazilian statistical data directly into R, including datasets related to agricultural production, land use, and economic indicators.
